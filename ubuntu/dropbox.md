### Dropbox

1. In terminal
  ```console
  cd ~ && wget -O - "https://www.dropbox.com/download?plat=lnx.x86_64" | tar xzf -
  ~/.dropbox-dist/dropboxd
  ```

1. Add Dropbox command to *Preferences/Startup Applicayions* in main Ubuntu menu:
  ```console
  /home/jeff/.dropbox-dist/dropboxd
  ```
